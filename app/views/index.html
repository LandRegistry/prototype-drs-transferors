Select application type prototype

This prototype focuses on the initial stage of the application process where users select the type of application they want to make. It aims to guide users to choose the appropriate application type based on their requirements, ensuring that applications are routed to the correct processing path, such as register update, transfer of part, LE, NL or JP1. The prototype explores two different designs and flows to achieve this goal effectively.

Transferor details prototype

This prototype is designed for user testing the redesign of the Transferor details section on DRS. It simulates three specific user journeys, focusing on reviewing and amending transferor details. Please note that some links and functionalities outside of these specific journeys may not be operational.
